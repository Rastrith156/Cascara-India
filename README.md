# Cascara India ☕🌿

**India's First Premium Coffee Husk Tea Brand**

Cascara India is an e-commerce storefront for premium dried coffee cherry husks sourced directly from the Western Ghats (Chikkamagaluru). This project is a front-end showcase featuring a dark, premium aesthetic, seamless animations, and a fully functional frontend shopping experience.

## ✨ Features

- **Multi-page Architecture**: Separated layouts for Products, Brew Guide, and Reviews for a clean navigational experience.
- **Frontend Shopping Cart**: A functional cart drawer powered by vanilla JavaScript and `localStorage`.
- **Demo Checkout Gateway**: Complete simulated checkout flow calculating dynamic pricing and shipping details.
- **Responsive Design**: Flawless layout on both desktop monitors and mobile devices without using external frameworks.
- **Premium Aesthetics**: Engineered with deep rich blacks and subtle gold/teal accents for a high-end feel.

## 🚀 How to Run Locally

You don't need any special developer tools or servers to run this site:
1. Clone or download the repository.
2. Open `index.html` directly in your web browser of choice.
3. Start exploring and test adding to your cart!

## 🛠️ Built With

- **HTML5**
- **Vanilla CSS3** (No Tailwind or external component libraries)
- **Vanilla JavaScript (ES6)** (Local state management, dynamic DOM rendering, Interaction observers)

## 📦 Project Structure

- `index.html` - The Home Page (Hero & Our Story)
- `products.html` - The Product Showcase array
- `brew.html` - Interactive steep guide
- `reviews.html` - Customer feedback & dynamically rendered reviews
- `checkout.html` - The demo Payment Gateway page
- `app.js` - Global functions, interactions, and `localStorage` cart handling
- `style.css` - Global design tokens and layout styling
